Microsoft Money 98 Financial Suite Trial Version

The Money 98 Financial Suite Trial Edition contains everything found in the full retail version of the product, with the following limitations:

  • The Trial Edition will only allow you to enter transactions within a 90-day window. When you upgrade to the full retail release of the Money 98 Financial Suite, you’ll be able to open and view all data entered in this version. (Note: data created in the Trial Edition cannot be opened by International Versions of Money 98.)
  • Certain files required to utilize online banking and bill-payment via the Internet are missing. North American customers may download these files - please see the Online Banking area for detailed instructions.
  • Due to size constraints, Money 98’s multi-media Product and Feature Tours are not available in this Trial Edition.

You can find additional information about this product at the Microsoft Money Website at http://www.microsoft.com/money/

To use the Money 98 trial version your computer must meet the following minimum system requirements:

Computer: 486/50 or higher processor (Pentium 90 or higher recommended)
Memory: 8 MB of memory for Microsoft Windows 95 (16 recommended )
Hard-Disk: 25-55 MB of available hard-disk space
Operating System: Microsoft Windows 95 or later
Display: VGA (Super VGA 256+ Color recommended)
Sound: Sound card with headphones or speakers recommended
Input Device: Microsoft mouse or compatible pointing device recommended
Optional: Microsoft Internet Explorer 3.02 (included)
9600 or higher-baud modem & Internet access required to use online features (28,800 baud recommended).
