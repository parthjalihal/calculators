Microsoft® IntelliMouse™

IntelliMouse Pointing device puts the power to zoom and scroll at your fingertips with its innovative Wheel. Get more done with less effort.

Quickly and easily scroll through Microsoft Word 97 documents, internet pages, Microsoft Excel spreadsheets, and more by pressing the Wheel while moving the mouse forward or back. Or you can move a few lines at a time simply by rolling the Wheel. Reviewing long documents is easy with the AutoScroll feature, which scrolls through your document automatically at whatever speed you choose. All of this functionality comes with the same comfort and precision of the popular Microsoft Mouse.

System Requirements

or®
To use Microsoft IntelliMouse as a two-button mouse, you need one of the following:
  • Windows 95 or later, Windows version 3.1 or later, MS-DOS® version 3.1 or later, Windows NT® Server version 3.51 or 4.0, or Windows NT Workstation version 3.51 or 4.0 operating systems (will not run on earlier versions of Windows or Windows NT)
  • Round mouse port (PS/2) or 9-pin serial port (coupon included for 25-pin serial port adapter)
To use the Microsoft IntelliMouse Wheel, you need:
  • IntelliPoint version 2.0 software (included)
  • One of the following: Windows 95, Windows NT Server version 3.51 or 4.0, or Windows NT Workstation version 3.51 or 4.0 operating systems
  • An application that works with the IntelliMouse Wheel (see product carton for partial list of applications)
  • 3.5" high-density disk drive
