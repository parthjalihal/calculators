The VSD2K.EXE file contains support for 2K Media on MO device in Windows 98.

2K media can be found on removable media devices used by Fujitsu & Sony.

To add support for 2K media in Windows 98:

1) Run the VSD2K.EXE from within Windows 98.
2) After the installation as been completed, 
   select "Yes" when requested to reboot your system.
