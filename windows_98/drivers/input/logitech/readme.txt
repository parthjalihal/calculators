The drivers found in this directory are for Logitech gaming devices. 

To install:

  1) Start -> Settings -> Control Panel
  2) Click on "Game Controllers"
  3) Click on the "Add..." button
  4) Click the "Add Other..." button
  5) Click on "Have Disk"
  6) Enter complete path to this directory, e.g. d:\drivers\input\logitech
  7) Click on OK to install
