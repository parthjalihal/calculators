The driver found in this directory is for Mouse Systems Pointing devices.

To install:

  1) Start -> Settings -> Control Panel
  2) Click on "Add New Hardware".
  3) Click on the "Next" button twice.
  4) After the Plug n' Play system check, select "No, the device isn't in the list." button.
  5) Click on "Next".
  6) Select "No, I want to select the hardware from a list" option. 
  7) Choose the "Mouse" category from the list presented and select "Next".
  8) Select the "Have Disk..." button.
  9) Enter complete path to this directory, e.g. d:\drivers\input\mousesys
  10) Click on OK to install
