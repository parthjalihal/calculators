Intercast for Windows98 provides enhancements to WebTV for Windows98 for
intercast aware television stations.

To install Intercast for Windows98, follow these instructions:

PLEASE NOTE: You must have WebTV for Windows98 installed and working prior
	     to following these procedures.

To install Intercast for Windows98:

  1) Run the "iit22020.exe" file in the \drivers\intercst directory.
  2) Follow the installation instructions contained in the setup program.
  3) If prompted to reboot your system, select "Yes".
