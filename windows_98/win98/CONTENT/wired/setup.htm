SETTING UP THE SCREENSAVER
You can get the Wired screen saver to run without interruption by tweaking the controls of the beta release of Internet Explorer 4.0. Why does it take 10 steps? This is because Internet Explorer 4.0 does not currently allow "one-butoon configuration" of screen savers. This feature will be included in the final release this fall. Please try it and tell us how it works for you!

1 Click on your Windows "Start" button. The Start Menu will pop up.

2 Select "Settings". The Settings menu will pop up.

3 Select "Control Panel". The Control Panel will pop up.

4 Double-click on the "Display" icon. The Display dialog box will pop up.

5 Click on the "Screen Saver" tab. The Screen Saver folder will pop to the front.

6 Click on the "Screen Saver" menu arrow.

7 Click on "Active Desktop Screen Saver".

8 Click the "Settings..." button. The Screen Saver Properties menu appears.

9 Select the "Wired Channel". (Note: Any other channels you select will also appear in the screen saver, popping up at intervals that you set in the Screen Saver properties menu also. If you have multiple screen savers selected, we recommend setting intervals of at least 30 seconds.) Click OK to save setting.

10 Click OK in the Display dialog to set the Wired screen saver.
